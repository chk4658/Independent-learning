# vue中过滤器的使用

Vue.js允许自定义过滤器，可用作一些常见的文本格式化。可以用在两个地方：**插值表达式**和**v-bind表达式**

### 全局过滤器的使用

```javascript
//过滤器中的function， 第一个参数已经规定死，永远都是过滤器管道前面传递过来的数据
Vue.filter('过滤器的名称',function(){})
//过滤器可以多次的调用
{{name | 过滤器的名称}}
```

### 定义私有的过滤器

```javascript
var vm = new Vue({
    el:'#app',
    methods:{},
    filters:{
        过滤器名称:function(){}
    }
})
```

过滤器调用的时候  采用就近原则

如果全局和私有过滤器名称一致，优先使用私有过滤器
