# 权限修饰符

- `public` 当前类，相同包下不同的类,不同包下的类
- `default` 当前类，相同包下不同的类，当前包下使用
- `private` 当前类
- `protected` 当前类，相同包下不同的类，让子类对象使用

## 修饰符小结

|             | 类   | 成员变量 | 成员方法 | 构造方法 |
| ----------- | ---- | -------- | -------- | -------- |
| `public`    | √    | √        | √        | √        |
| `default`   | √    | √        | √        | √        |
| `protected` |      | √        | √        | √        |
| `private`   |      | √        | √        | √        |
| `abstract`  | √    |          | √        |          |
| `static`    |      | √        | √        |          |
| `final`     | √    | √        | √        |          |
